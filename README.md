Retail Sales & Inventory Analysis Dashboard (Power BI)

Project Overview
This project is an interactive Retail Sales Dashboard built using Power BI to analyze sales performance, inventory status, and product demand across regions and time.

The dashboard helps in understanding business performance and supports data-driven decision-making.

How the Data Works

The dataset represents a retail business and includes:

Product Details: Product name and category such as Electronics, Furniture, Accessories, and Stationery
Sales Data: Units sold, unit price, and total sales
Time Data: Month-wise and year-wise sales
Region Data: East, West, North, and South regions
Inventory Data: Reorder status based on stock levels

Data Processing
The raw data was cleaned and transformed using Power Query.
DAX was used to create calculated measures like Total Sales, Total Units Sold, and Reorder Count.
Relationships between tables were built to enable filtering and accurate analysis.
